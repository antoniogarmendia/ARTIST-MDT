SQLDiscoverer, Version 1.0.0

=================================

ABOUT
-----
- SQLDiscoverer, component from the ARTIST Model Discovery Toolbox, allows automatically 
producing an SQL model from initial SQL source code. The obtained SQL model covers the 
'INSERT' command.

=========================================================================================

INCLUDED FUNCTIONALITIES
------------------------
- Each INSERT commands are injected into the SQL model.
- This component currently considers:
	* As input: SQL instructions
	* As output: an SQL model generated out of the input.
