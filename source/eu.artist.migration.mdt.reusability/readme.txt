 Tool: Reusability Trace Tool (RTT)
 Plugins: eu.artist.migration.mdt.reusability
 
 Description:The “Reusability Trace Tool” (or RTT) is a tool whose objective is 
 1) to provide a first automatic selection of pieces of code that can be transferred directly to the target system in the modernization process and 
 2) to inject that code directly once the user approves it..
 
 Usage: Install the plugin in the eclipse environment and select the new option that appears in the main menu. 
 After that, follow the steps described on section 3.3.4 of public deliverable D8.2.2 or ARTIST project (http://www.artist-project.eu/)
