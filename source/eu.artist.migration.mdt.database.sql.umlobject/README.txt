SQL2UMLObject, Version 1.0.0

=================================

ABOUT
-----
- SQL2UMLObject, component from the ARTIST Model Discovery Toolbox, allows automatically 
producing an UML Object model from an initial SQL model and an UML model. The obtained 
UML Object model covers the classes instantiated.
This component is complementary to the SQLDiscoverer component from the ARTIST Model 
Discovery Toolbox.

=========================================================================================

INCLUDED FUNCTIONALITIES
------------------------
- Each Classes and Tables are transformed into an UML object.
- This component currently considers:
	* As input: SQL model containing the values of the classes to instantiate and an
	UML model, containing the classes to instantiate.
	* As output: an UML Object model generated out of the input.
